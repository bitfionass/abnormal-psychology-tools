# 变态心理学辅助智能工具站

统一收纳变态心理学课程的课堂交互工具与诊断训练工具。

## 页面入口

- 首页：`index.html`
- DSM-5诊断标准速查手册：`dsm5-handbook.html`
- 变态心理学案例库：`case-library.html`
- 模拟咨询训练系统：`counseling-simulator.html`
- 案例随机分配工具：`case-randomizer.html`
- W14个案概念化练习：`w14-case-conceptualization.html`

注意：这些页面仅用于教学训练，不用于真实临床诊断。
